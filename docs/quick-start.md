## 🎯 立即启动项目

现在你可以正常启动项目了！

### 方法1: 分别启动（推荐用于开发调试）

**启动后端：**
```bash
cd backend
npm run dev
```
后端将运行在 http://localhost:3001

**启动前端：**
```bash
cd frontend  
npm run dev
```
前端将运行在 http://localhost:3000

### 方法2: 同时启动

```bash
# 在项目根目录
npm run dev
```
